== Description ==

**English:**
Gift-i-Card integrates WooCommerce with the Gifticard.pro service for international users. This plugin allows global WooCommerce stores to sell and manage gift cards easily and securely via [gifticard.pro](https://gifticard.pro).

**فارسی:**
افزونه «گیفتی کارت» فروشگاه ووکامرس شما را به سرویس گیفتیکارت متصل می‌کند و ویژه کاربران ایرانی است. با این افزونه می‌توانید به‌سادگی و با امنیت بالا، انواع گیفت کارت را از طریق [gifticard.ir](https://gifticard.ir) در فروشگاه خود ارائه دهید.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/gift-i-card` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Gift-i-Card menu to configure the plugin
